# ASP.NET Core 局域网文件共享站点

一个基于 **ASP.NET Core** 的轻量级文件共享站点，用于在 **局域网（LAN）环境**中快速传输文件。
无需第三方服务，部署简单，适合临时文件交换、内网协作、测试环境文件分发等场景。

------

## ✨ 功能特性

- 📁 文件浏览与下载
- ⬆️ 支持文件上传（可选）
- ⚡ 基于 HTTP 的高速局域网传输
- 🖥️ 跨平台运行（Windows / Linux / macOS）
- 🧩 依赖极少，开箱即用
- 🔐 可扩展访问控制（如 IP 限制、简单鉴权）

------

## 🛠 技术栈

- **.NET**：.NET10 
- **Web 框架**：ASP.NET Core
- **前端**：原生 HTML 
- **运行方式**：Kestrel 自托管

------

## 🚀 快速开始

### 1️⃣ 直接双击运行-（win64）

### 2️⃣ 克隆运行项目

```bash
dotnet run
```

默认启动后，在浏览器中访问：

```
http://localhost:9000
```

或使用局域网 IP 访问，例如：

```
http://192.168.1.100:9000
```

------

## 📂 文件存储说明

- 默认共享目录：

  ```
  ./FILES
  ```

- 支持大文件传输

## 🧠 适用场景

- 局域网内快速传输大文件

## ⚠️ 注意事项

- 本项目**默认不适合暴露到公网**
- 局域网内使用体验最佳
